# 基于开题答辩PPT的 6 周逐日任务清单（Notion / Typora 美化版）

> 适用对象：基于 **SegFormer 基线 → 结构化剪枝 → FP16 优先量化 → TensorRT 部署 → Jetson Orin NX 端侧验证 → 泛化测试** 路线推进毕业设计/论文工作的研究生。  
> 使用方式：可直接粘贴到 **Notion / Typora / Obsidian / VS Code Markdown** 中使用。  
> 建议执行周期：**6 周**；若时间紧张，可参考文末 **4 周压缩版**。  
> 建议日均有效科研时长：**5–6 小时**；高强度实验日 **6–8 小时**；整理复盘日 **3–4 小时**。

---

## 一、总目标

### 1. 研究主线

- [ ] 完成 **RailSem19 数据集** 的训练、评估与可视化整理
- [ ] 复现并稳定训练 **SegFormer 基线模型**
- [ ] 基于 **DepGraph / Isomorphic Pruning** 完成结构化剪枝实验
- [ ] 优先完成 **FP16** 量化与部署验证，必要时补充 **INT8**
- [ ] 完成 **ONNX → TensorRT** 推理部署
- [ ] 在 **Jetson Orin NX** 上完成端侧测速与效果验证
- [ ] 完成泛化测试、失败案例分析和论文实验章节整理

### 2. 阶段指标

- [ ] 参数压缩率达到 **40% 以上**
- [ ] 端侧推理速度达到 **30 FPS 以上**
- [ ] 单帧延迟控制在 **33 ms 以内**
- [ ] 常规场景精度下降不超过基线 **10%**
- [ ] 至少形成一套可直接写入论文与答辩 PPT 的完整实验结果
